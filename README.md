# Cisco ASA Firewall Configuration

A networking and infrastructure project focused on configuring and troubleshooting enterprise-style network communication using Cisco ASA Firewall, multilayer switching, VLAN segmentation, NAT, ACLs, and OSPF routing.

The project was developed in a virtualized lab environment to simulate secure inter-network communication across multiple subnets while implementing routing, firewall security policies, and traffic filtering workflows.

# Key Features

* Cisco ASA firewall configuration
* VLAN segmentation and inter-VLAN routing
* NAT configuration for internal-to-external communication
* ACL-based traffic filtering and security rules
* OSPF routing protocol setup
* Multilayer switch configuration
* Static and dynamic routing
* Subnet-based network design
* Connectivity troubleshooting and packet flow verification

# Technologies Used

* Cisco ASA Firewall
* Cisco Routers & Switches
* OSPF
* VLANs
* NAT
* ACLs
* Multilayer Switching
* IP Routing
* Subnetting

# Project Overview

The lab environment consisted of multiple isolated network segments connected through a multilayer switch and secured using Cisco ASA firewall policies. OSPF routing was configured between routers while VLAN-based segmentation enabled controlled communication between internal networks.

Firewall rules and ACL policies were implemented to manage inbound and outbound traffic flows, while NAT configuration enabled secure communication between internal and external network zones.

The project also involved troubleshooting routing behavior, verifying end-to-end connectivity, analyzing packet flow, and validating network communication across multiple subnets.
